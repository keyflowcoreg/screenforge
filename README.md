# ScreenForge

AI-powered App Store & Play Store screenshot generator.

[![Live Demo](https://img.shields.io/badge/Live_Demo-screenforge.vercel.app-blue?style=for-the-badge)](https://screenforge-ten.vercel.app)

[Live Demo](https://screenforge-ten.vercel.app) | [Report Bug](https://github.com/keyflowcoreg/screenforge/issues)

## Features

- Upload raw screenshots, get professional marketing images
- Device frames for iPhone, iPad, and Android
- Multiple style presets: gradient, minimal, dark, colorful
- All App Store and Play Store sizes included
- One-time $19 payment via USDC on Base

## Tech Stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS v4 · Framer Motion · x402 Payments

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:4010](http://localhost:4010) to view it locally.
